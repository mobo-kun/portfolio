# Behdad Morsalpoor — Personal Portfolio

> **Live site → [behdad.studio](https://behdad.studio)**

A personal portfolio website for **Behdad Morsalpoor**, Senior Product Design Lead specialising in Fintech, Crypto, and behavioural design. Built entirely with Claude.

---

## What's inside

- **Hero** — Animated profile orb with rotating conic-gradient border and floating tool icons
- **Case Studies** — Featured work with 3D tilt cards (Notion CMS, coming soon)
- **CV** — Full timeline of professional experience
- **Recommendations** — Infinite auto-scrolling marquee carousel
- **About** — Bio, contact links, and social channels
- **Blog** — Writing and Substack articles (coming soon)

## Tech stack

| Layer | Choice |
|---|---|
| Framework | Next.js 16 (App Router) |
| Styling | Tailwind CSS v4 |
| Animations | Framer Motion |
| Fonts | Intel One Mono + Roboto Mono |
| Theme | Light / Dark (next-themes) |
| Deployment | Vercel |

## Run locally

```bash
npm install
npm run dev
# → http://localhost:3000
```

## Build

```bash
npm run build
npm run start
```
