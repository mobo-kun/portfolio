# Tool Icons

Drop 64×64 SVG files here. Each file name must match the `label` used in `HeroSection.tsx`.

| File name       | Tool     |
|-----------------|----------|
| `figma.svg`     | Figma    |
| `cursor.svg`    | Cursor   |
| `claude.svg`    | Claude   |
| `grok.svg`      | Grok     |
| `gemini.svg`    | Gemini   |
| `podcast.svg`   | Podcast  |

Files placed here are automatically served at `/tool-icons/<name>.svg` and picked up by the floating icons without any code changes.
